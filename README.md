# css-draw

Awesome icon for modern webapps drawed by css.

## Base icons

* circle
* rect
* trangle
* leftArrow
* rightArrow
* plus
* minus
* heart
* solidHeart
* navLeft
* navRight
* close
* rectCircle
* envelope
* pencil
* glass
* search
* asterisk

## Usage

```html
<i data-icon='circle' class="css-icon"></i>
```

```js
cssDraw.render({
	shadowDom: true 	// if set shadowDom, css icon will be set in shadow dom.
});
```

## API

* render
* extend

* `npm run dev`
